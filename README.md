# 🤖 AI Gesture-Controlled App Launcher

A real-time AI system that recognizes hand gestures using a webcam and launches desktop or web applications without using a mouse or keyboard.

## 🚀 Features
- Real-time hand gesture recognition (MediaPipe + Deep Learning)
- Gesture-based desktop app launching (VS Code, Excel, Spotify)
- Gesture-based web app access (YouTube, Gmail, GitHub, LinkedIn)
- Cooldown & edge-triggered logic to prevent duplicate actions
- Runs fully on CPU (no GPU required)

## 🧠 Technologies Used
- Python 3.10
- TensorFlow / Keras
- MediaPipe
- OpenCV
- NumPy
- Scikit-learn
